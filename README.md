# os-titlecard
## a minimal TTY titlecard

requires pyfiglet and printy, not sure the current setup will directly install them

# installation
## cloning:
```
> git clone https://github.com/scringlebug/os-titlecard.git
> pip install .
```
## directly:
(not sure this will work)
```
pip install git+https://github.com/scringlebug/os-titlecard.git
```
